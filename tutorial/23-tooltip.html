Tooltip Example

To create a tooltip, add the data-bs-toggle="tooltip" attribute to an element. Use the title attribute to specify the text that should be displayed inside the tooltip.
Note: Tooltips must be initialized with JavaScript to work.

Tooltip Positioning

The data-bs-placement attribute specifies the tooltip position.
